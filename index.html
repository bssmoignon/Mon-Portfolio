Projets

MedTech HA sur AWS

Projet AWS : Déploiement d’une application MedTech hautement disponible Contexte : Dans le cadre de ma formation en virtualisation et cloud computing, j’ai travaillé sur le déploiement d’une application web pour une startup fictive, MediConnect, spécialisée dans la santé. L’objectif était de garantir une haute disponibilité et une sécurité optimale pour les patients et le personnel médical. Objectif du projet : Mettre en place une infrastructure cloud sur AWS pour héberger une application web critique. Assurer la haute disponibilité, la résilience et l’élasticité de l’application. Optimiser les coûts et la gestion des ressources cloud. Détails techniques : Création et configuration de instances EC2 pour le serveur web. Mise en place de load balancer (ELB) pour répartir le trafic et éviter les interruptions. Utilisation de RDS pour la base de données relationnelle, avec sauvegarde et réplication. Implémentation de Groupes de sécurité et IAM pour sécuriser l’accès. Surveillance et gestion via CloudWatch pour assurer la performance et détecter les incidents. Résultat : L’application est hautement disponible et peut supporter des pics de trafic. Les données des patients sont sécurisées et sauvegardées automatiquement. L’infrastructure est scalable, permettant de gérer la croissance future de l’entreprise. Compétences acquises : Maîtrise des services AWS (EC2, RDS, ELB, CloudWatch, IAM). Gestion de la haute disponibilité et de la résilience. Optimisation des ressources cloud et sécurité des applications web.
